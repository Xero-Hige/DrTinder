External libraries¶

Dr. Tinder depends on the following external libraries. Buildscript downloads all dependencies automatically, but you should check that this libraries projects are still compatible with the source version before building. Bellow all libraries are listed including use and project url.

(Higesoft nor any of the developers of Dr. Tinder are related to any of this projects. The maintenance of these libraries is the sole responsibility of their respective original authors)

Android Shape Imageview¶

Description:

Custom shaped Android imageview components
Project Url:
  • https://github.com/siyamed/android-shape-imageview

Circular Imageview¶

Description:

Custom view for circular images in Android while maintaining the best draw performance
Project Url:
  • https://github.com/Pkmmte/CircularImageView

Firebase CM¶

Description:

Deliver and receive messages across platforms reliably
Project Url:
  • https://firebase.google.com/docs/cloud-messaging/

Glide¶

Description:

An image loading and caching library for Android focused on smooth scrolling
Project Url:
  • https://github.com/bumptech/glide

Guava¶

Description:

Google Core Libraries for Java 6+
Project Url:
  • https://github.com/google/guava

OpenCSV¶

Description:

Opencsv is a very simple CSV (comma-separated values) parser library for Java.
Project Url:
  • http://opencsv.sourceforge.net/
  • https://sourceforge.net/projects/opencsv/

Retrolambda¶

Description:

Backport of Java 8’s lambda expressions to Java 7, 6 and 5
Project Url:
  • https://github.com/orfjackal/retrolambda
Gradle Plugin:
  • https://github.com/evant/gradle-retrolambda

Spring¶

Description:

Core support for dependency injection, transaction management, web applications, data access, messaging, testing and more.
Project Url:
  • https://projects.spring.io/spring-framework/
  • https://github.com/spring-projects/spring-framework

Swipe-Deck¶

Description:

A Tinder style Swipeable deck view for Android
Project Url:
  • https://github.com/aaronbond/Swipe-Deck
