API¶

Login¶

URL http://<server>/user
HTTP VERB POST
PARAMETROS

Headers

Authorization: username=”mi_username” pass=”mi_contraseña”

Body

localization=0.125,1.25
RESPUESTA HTTP 200 | HTTP 401 (login incorrecto)
